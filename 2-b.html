Section 01

目線とは何か

「目線」とはトレンドの方向に対する、構造的な判断だ。「なんとなく上っぽい」という感覚とは違う。波の構造から導き出される、論理的な立場のこと。
トレンド定義目線
上昇トレンド波の高値・安値が連続して切り上がっている状態上目線
下降トレンド波の高値・安値が連続して切り下がっている状態下目線
目線の重要性は、「変わらない間は変えない」という点にある。上昇トレンドの最中に陰線が何本続こうと、目線はずっと上のまま。それが構造的な判断というものだ。

Section 02

目線が変わる条件はひとつだけ

UTにおける目線転換の条件は、シンプルに定義されている。
RULE — DIRECTION CHANGE

上昇目線が変わる条件：押し安値が割れること。
下降目線が変わる条件：戻り高値が更新されること。

それ以外の条件はない。「なんとなく下に向かった気がする」は目線転換の根拠にならない。

ここで「押し安値」という概念が登場する。これがA-3で学んだ「頂点の確定」と深く結びついている。
ただ、多くの人がここでつまずく。どれが押し安値なのか、という問題だ。
下の図を見てほしい。a・b は押し安値。けれど x・y も安値なのに、押し安値とは呼ばない。なぜか。
x・y は、①②という上昇波を構成している内部の小さな揺れにすぎない。①②という波そのものがまだ崩れていない以上、その内部の x・y は、この粒度では押し安値に「昇格」しない。押し安値とは、判断レベル（ローソク波）の波が押した安値のこと——2-Aで学んだ粒度の固定が、ここで効いてくる。

「これはトレンド転換じゃないか？」と感じるのは自然だ。でも問いはひとつ——押し安値を割ったか？

B-2｜押し安値・戻り高値の確定
CHAPTER B-2 — CONFIRMATION
B-2｜押し安値・戻り高値の確定
押し安値も戻り高値も、確定するまでは「候補」でしかない。そして確定は、いつも後からやってくる。この逆説を掴むと、目線の判断が一段安定する。
Mori
B-1で「目線が変わる条件は押し安値が割れること」を確認した。

でも——その押し安値は、いつ"押し安値"になるのか。ここを正確に掴んでいないと、目線の判断が根本からズレてしまう。

少し不思議な話をする。一度腑に落ちると、チャートの見え方が変わるはずだ。
Section 01

押し安値の確定順序

まず上昇方向で考える。値動きの時系列はこうだ。
①
上昇（a → b）が起きる
↓
②
b から押しが入り、安値 d が形成される（この時点ではまだ「直近安値」）
↓
③
d から反発した c が、b の高値を上抜ける
↓
④
この瞬間に初めて、過去の d が「押し安値」として確定する
Fig.01 — c が b を上抜けた瞬間、過去に存在していた d が「押し安値」として確定する
THE PARADOX OF CONFIRMATION

d は c より時間的に先に存在している。
しかし認識として確定するのは、c の後だ。

未来の高値が、過去の安値を確定させる。リアルタイムでは、すべての安値は「候補」でしかない。

これはチャートの不思議ではなく、認識の構造だ。「あの安値が押し安値だった」という事実は、その後のブレイクが起きて初めて成立する。
この"未来が過去を確定させる"逆説は、初めてではない。A-3「頂点の確定」で扱ったものと、同じだ。あのとき頂点で見たことが、ここでは押し安値で起きている。
Section 02

戻り高値の確定——完全な鏡像

下降方向では、構造がそっくり裏返る。やっていることは押し安値と同じだ。
①
下落（a → b）が起きる
↓
②
b から戻りが入り、高値 d が形成される（この時点ではまだ「直近高値」）
↓
③
d から再び下落した c が、b の安値を下抜ける
↓
④
この瞬間に初めて、過去の d が「戻り高値」として確定する

確定の対象確定条件
押し安値（上昇方向）その後の上昇が直前の高値を上抜けた瞬間
戻り高値（下降方向）その後の下落が直前の安値を下抜けた瞬間
CS
用語の精度の話をさせてほしい。

「押し安値」「戻り高値」はどちらも、確定した後にしか使えない言葉だ。反発して戻ってきた安値を見て「押し安値だ」と言いたくなる場面があるが——高値ブレイクが起きるまでは「直近安値」または「押し安値候補」が正確だ。

言葉の精度が、そのまま判断の精度になる。
CS Note — 用語の使用基準
確定前 →「直近安値／直近高値」または「〜候補」。
確定後 →「押し安値／戻り高値」。
下から戻ってきた高値を、確定前に「戻り高値」と呼ばない。
Section 03

目線判断と確定の関係

B-1で「押し安値が割れたら目線が変わる」と学んだ。B-2の視点から見直すと、ひとつ重要なことが見えてくる。
押し安値は確定した後にしか「押し安値」と呼べない。つまり、目線判断の根拠にできる安値は、そもそも確定済みのものだけだ。
Key Point
目線判断の根拠に使える安値は、確定済みの押し安値だけ。

「あそこを割ったら目線が変わる」と言えるのは、その安値が押し安値として確定した後だけだ。確定前の候補に依存した判断は、構造的に不安定になる——これが「目線が迷う」原因のひとつになっている。
Mori
「未来が過去を確定させる」——これはチャートだけの話じゃない。

ある動きが押し目だったか、転換だったか。それも後から確定する。

だから、確定前に決め打ちして焦らなくていい。構造が確定するのを、静かに待てばいい。
Summary

CHAPTER B-3 — GRANULARITY
B-3｜粒度の固定
1枚のチャートには、粒度の違う「押し安値」が同時に存在している。 どの押し安値を判断基準にするかを決めて、そこに留まり続ける——それが粒度の固定だ。 このセクションでは、実際のチャートで「目視が下を向いても判断が揺るがない」とはどういうことかを、最後まで追いかける。
Section 01

押し安値は、粒度ごとに存在する

同じチャートでも、見ている粒度が変われば「押し安値」は別の場所になる。 小さな上昇には小さな押し安値があり、大きな上昇には大きな押し安値がある。 この2つは同時に走っていて、しかも割れるタイミングがずれる。ここに迷いの種がある。
CS
このチャートで起きていることを、順に追う。 B から C への上昇——これを上昇アと呼ぶ。この上昇を支えている押し安値は B だ。ローソク波レベルで「上昇が生きているか」を判断する基準は、ただ一点、Bが割れたか。これだけ。 途中の安値 x は、B→C をつくっている"もっと小さな上昇"の押し安値だ。粒度がひとつ下。 さて、C のあと y が x を割ってきた。ここで小さな上昇は転換する。多くの人が最初に揺れるのがこの瞬間だ——「もう下かな？」 だが話は終わらない。y からの戻りは C を越えられず、切り下げ高値 C' をつくる。そして z へ。C→y→C'→z——切り下げの高値と安値が揃い、目視レベルでは立派な下降トレンドが"完成"する。ここでほとんどの人が確信する。「天井は C で決まった。目線は下だ」 ——これが、粒度の混在の正体だ。 下位粒度（x・C'・z）で起きている下降を、上位粒度（B）の判断にまで持ち込んでしまっている。だが上位粒度の判断基準は最初から変わっていない。Bが割れたか。そして B は、一度も割れていない。 結末はチャートの右側にある。B が守られたまま買いが戻り、E が C を上抜けた。上昇は続いていた。天井だと思われた C はただの通過点で、下降だと思われた z は——次の押し安値 D に昇格した。

Key Point — B-3のまとめ
① 1枚のチャートに、粒度の違う押し安値が同時に存在する（小＝x ／ 大＝B） ② 粒度の固定とは、判断する粒度を決め、その押し安値だけを基準にすること ③ 下位粒度の下降（x割れ・C'・z）は「見る」が、上位粒度（B）の判断には入れない ④ B が割れない限り上昇は生きている——E が C をブレイクし、z は新しい押し安値 D へ
C→y→C'→z のように"転換したと見せて戻される"動きは、後の章でユニバーサルパターンとして正面から扱う。 今は「粒度を固定すれば飲まれない」——それだけ持って帰ればいい。
